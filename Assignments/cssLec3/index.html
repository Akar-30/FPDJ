Homework Assignment: CSS Grid, Flexbox, and Styling Lists

Sections:

  • CSS Grid Layout
  • Flexbox Layout
  • Styled List

Part 1: CSS Grid Layout

Objective: Create a web page layout using CSS Grid with the following sections: A header that spans the entire width of the page, a sidebar on the left side that is 200px wide, a main content area that takes up the remaining width, and a footer that spans the entire width of the page.

Requirements: The layout should have a 10px gap between the grid items. The sidebar should have a fixed width of 200px, while the main content area should be responsive and take up the rest of the space. The header and footer should be full-width.

Bonus: Add another section in your layout for "Related Articles" to the right of the main content (use CSS Grid to achieve this).

Part 2: Flexbox Layout

Objective: Create a responsive navigation bar using Flexbox with the following elements: A logo on the left side, three navigation links (Home, About, Contact) in the middle, and a search box aligned to the right.

Requirements: The navigation links should be evenly spaced horizontally. The search box should align to the right side of the navbar. The navbar should collapse into a column layout on smaller screens (under 600px width).

Part 3: Styling a List

Objective: Create a styled list with the following features: An unordered list (ul) with at least 5 list items. Customize the bullet points by replacing them with an image or a custom character. Add padding and a background color to each list item. When you hover over each list item, change the background color and font style (e.g., italic or bold).

Bonus: Create a second list (ordered list) and style the list numbers to have a different font, size, and color than the list text.

Submission Requirements:

  • Part 1 (CSS Grid): Create an HTML and CSS file (grid-layout.html, grid-layout.css) for the grid layout.
  • Part 2 (Flexbox): Create an HTML and CSS file (flexbox-navbar.html, flexbox-navbar.css) for the Flexbox navigation bar.
  • Part 3 (List Styling): Create an HTML and CSS file (styled-list.html, styled-list.css) for the styled list.

Resources to Review:

  • CSS Grid: Review how to use grid-template-columns, grid-template-rows, grid-template-areas, and grid-gap.
  • Flexbox: Review properties like display: flex, justify-content, align-items, and flex-direction.
  • List Styling: Review how to style lists using properties like list-style, list-style-image, padding, background-color, and hover effects.
